概要 (The idea)

I've always wanted to save my secret information (such as passwords, two-factor authentication backup codes, etc.) safely on the Web. I'm sure I will easily forget where and how I stored the secret information, so I decided to put all my secret information here on the Web so that I can find this page on Google. After visiting this page, I can access all my secret information using the EpisoPass system, which allows me to retrieve secret information only by answering secret questions.
