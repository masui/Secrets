増井の秘密情報

Toshiyuki Masui's secret information

概要 (The idea)

I want to save my secret information such as passwords, 2-factor authorization backup codes, etc. on the Web. I know that I will easily forget where and how I put the secret information, so I decided to put all my secret information here on the Web so that I can find them on Google. I can access all the secret information using the EpisoPass system, with which I can retrieve secret information only by answering to secret questions.
